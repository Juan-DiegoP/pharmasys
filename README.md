# PharmaSys

## Descripción
PharmaSys es una aplicación desarrollada en Python utilizando la libreria Tkinter.  
El sistema permite gestionar información básica de una industria farmacéutica mediante una interfaz gráfica organizada en pestañas.

## Tecnologías utilizadas
- Python
- Tkinter

## Módulos del sistema
El sistema está dividido en cuatro módulos:

1. Principios Activos  
2. Medicamentos  
3. Producción  
4. Control de Calidad  

## Ejecución del programa

1. Descargar o clonar el repositorio.
2. Abrir la carpeta del proyecto.
3. Ejecutar el archivo principal:

python pharmasys.py
